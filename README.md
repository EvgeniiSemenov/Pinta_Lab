Student project for learning mobile development with .NET MAUI.

## Features
- Create cabinet orders with configurable parameters  
- User registration and login
- Order history tracking
- Simple admin panel

## Tech Stack
- .NET MAUI
- SQLite database  
- C# & XAML

## Getting Started
1. Open PintaLab.sln in Visual Studio 2022+
2. Run on Windows emulator or device
